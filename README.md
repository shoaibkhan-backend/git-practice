# Git Practice Repository

This repository is created to practice Git and GitHub.

## Purpose
- Learn Git commands
- Practice GitHub workflow
- Understand version control system (VCS)

## Topics Covered
- Git initialization
- git status
- git add
- git commit
- git push
- git pull
- git clone
- git remote
- README and .gitignore usage
- Branching and merging
- Tags and releases

## Sample Files (for practice)
- README.md – File documentation
- sum.py    – Simple Python Program for Addition
- hello.py  – Simple python program hello
- armstrong.py – armstrong_number program 
- fibonacci_series.py – fibonacci series program python

## How to Run prgrams
- python sum.py
- python hello.py
- python armstrong.py

## Tools Used
- Git
- GitHub
- VS Code
- Git Bash

## Workflow
1. Edit or add files
2. Stage changes using `git add`
3. Commit changes using `git commit`
4. Push changes to GitHub using `git push`
